# MyLinkBox: 단일 파일 React 링크 관리 대시보드

## 🔗 프로젝트 소개

**MyLinkBox**는 사용자가 중요한 웹 링크를 한곳에 저장, 분류 및 관리할 수 있도록 설계된 효율적인 링크 관리 애플리케이션입니다. React와 Tailwind CSS로 프론트엔드를 구축하고, Google Firebase Firestore를 사용하여 실시간으로 데이터를 저장하고 동기화합니다.

이 프로젝트는 모든 기능을 단일 React 컴포넌트 파일 내에 통합하여 간결하고 독립적인 실행 환경을 제공하는 것을 목표로 합니다.

## ✨ 주요 기능

* **링크 관리 (CRUD):** 새로운 링크를 추가하고, 기존 링크를 열람 및 삭제할 수 있습니다.
* **실시간 대시보드:** 총 링크 수, 즐겨찾기 링크 수, 카테고리 수 등 주요 통계를 한눈에 볼 수 있습니다.
* **즐겨찾기 기능:** 중요한 링크는 즐겨찾기로 지정하여 별도로 관리할 수 있습니다.
* **태그 및 카테고리:** 링크에 태그와 카테고리를 지정하여 효율적인 검색 및 분류가 가능합니다.
* **사용자 인증:** Firebase를 통한 익명/커스텀 토큰 인증을 기반으로 안전하게 데이터를 저장합니다.


## 🛠️ 기술 스택

| 분류 | 기술 | 설명 |
| :--- | :--- | :--- |
| **Frontend** | React (Hooks) | 사용자 인터페이스 구축 및 상태 관리 |
| **Styling** | Tailwind CSS | 빠르고 반응성이 뛰어난 유틸리티 기반 CSS 프레임워크 |

## 🚀 사용 방법

### 1. 실행 및 접근

이 애플리케이션은 **단일 파일 React 컴포넌트**로 제공되어, Canvas 환경에서 별도의 설치 과정 없이 즉시 실행됩니다.

* `LinkVaultApp.jsx` 파일을 로드하면 자동으로 Firebase 인증 과정을 거쳐 대시보드가 표시됩니다.

### 2. 링크 추가

1.  대시보드 오른쪽 사이드바에 있는 **"새 링크 추가"** 버튼을 클릭합니다.
2.  모달 창에 URL, 제목, 태그(쉼표 구분), 카테고리를 입력합니다.
3.  **"링크 저장"** 버튼을 클릭하면 Firestore에 저장되며 대시보드에 실시간으로 반영됩니다.

### 3. 링크 관리

* **즐겨찾기:** 최근 링크 목록에서 북마크 아이콘을 클릭하여 링크를 즐겨찾기에 추가하거나 해제할 수 있습니다.
* **삭제:** 휴지통 아이콘을 클릭하여 링크를 삭제합니다. (삭제 전 확인 모달이 표시됩니다.)

### 4. 로그아웃

* 오른쪽 사이드바 하단의 **"로그아웃"** 버튼을 클릭하면 안전하게 세션이 종료되며 애플리케이션의 랜딩 페이지(` / `)로 이동합니다.
